Desarrolla una aplicación en Java por consola que permita al usuario calcular el área y el perímetro de diferentes figuras geométricas (círculo, rectángulo y triángulo). El programa debe estar estructurado utilizando los principios de la Programación Orientada a Objetos (POO) y debe aplicar los fundamentos de programación vistos en clase.

En esta lo primero hago las clases circulo rectangulo y triangulo equilatero con cada uno de sus atributos y ademas creo la clase figura que es de la que heredan las tres clases nombradas anteriormente, en cadda clase que hereden meto un contructor con sus getters y setters y overrrides para que sobreescriban a la clase forma que es una clase abstracta con metodos de calcular el area y el perimetro y me muestra un mensaje printf donde me mostratra el resumen de las figuras

en la clase main creo el escanner y dentro creo un array de figuras de un maximo de 20 donde se almacenaran, despues creo con un do y un switch una pantalla que me muestre todas las opciones que tengo para hacer dependeindeo dee lo que quiera,despues mas tarde en cada caso hago lo necesario para mostrar todos los metodos de cada clase y por ultimo con un try y un catch pongo los posibles errores que pueda haber y que me pide que meta.
